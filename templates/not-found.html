Multipass Instance Not Found

You're seeing this page because you're trying to access a Multipass instance through UnifyPass, but the instance wasn't found. This could be because:

  • The instance name in your URL doesn't match any running Multipass instance
  • The instance exists but isn't currently running
  • There was an error getting the instance's IP address
Quick Fix:
1. Check your running instances:
multipass list

2. Make sure your instance is running:
multipass start your-instance-name

URL Format:
The correct format is: http://instance-name.local
Replace "instance-name" with your actual Multipass instance name.
